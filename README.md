## ngCanvas

A angularJS directive for creating canvas in NG way. defined several directive tag for general shape and graphic.

### Sample Code

````
<ng-canvas>
    <nc-rect cstyle="canvasConfig.children.rectA">
    </nc-rect>
    <nc-circle cstyle="canvasConfig.children.rectA">
    </nc-circle>
</ng-canvas>
````
